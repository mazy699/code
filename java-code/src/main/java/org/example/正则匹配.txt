正则匹配三段式 xxx-xxx-xxx

^((?!-).)*[-]((?!-).)*[-]((?!-).)*$
或者
([^-]*)[-]([^-]*)[-]([^-]*)

以不包含“-”的任意字符开头（0个或多个）----------------------^((?!-).)*
匹配“-”------------------------------------------------------------------[-]

第二种可以通过
Pattern p = Pattern.compile(正则);
Matcher m = p.matcher(要匹配的字符串);
while(m.find()){
    m.group(1);
    m.group(2);
    m.group(3);
}
简单获取（）中匹配的值
